`const canvas = document.getElementById('landscape');` Verknüpft das ``-Element im HTML-Dokument mit der JavaScript-Logik. HTML5 Canvas, DOM-Element auswählen
`const ctx = canvas.getContext('2d');` Erstellt einen 2D-Kontext zum Zeichnen auf der Leinwand (`canvas`). 2D-Rendering, Canvas-API
`const button = document.getElementById('moveFlowers');` Verknüpft den Button mit dem JavaScript-Code, um ihn interaktiv zu machen. DOM-Element auswählen, Interaktivität
`canvas.width = window.innerWidth;`
`canvas.height = window.innerHeight;`		 Setzt die Größe der Leinwand auf die volle Fenstergröße. Responsive Design, Fenstergröße abfragen
`function drawSky() { ... }` Zeichnet den Himmel mit einem vertikalen Farbverlauf von Blau nach Weiß. Farbverlauf, `createLinearGradient`
`function drawGrass() { ... }` Zeichnet die Wiese als grünes Rechteck in der unteren Hälfte der Leinwand. Rechtecke zeichnen, Flächen füllen
`const flowers = [];`
`const clouds = [];`		 Arrays zur Speicherung der Position und Animationseigenschaften von Blumen und Wolken. Arrays, Animation
`function initFlowers() { ... }` Erstellt Blumen mit zufälligen Positionen und Schwankungseigenschaften. Zufall (`Math.random()`), Initialisierung
`function initClouds() { ... }` Erstellt Wolken mit zufälligen Größen, Geschwindigkeiten und Positionen. Zufall, Initialisierung
`function drawFlower(x, y) { ... }` Zeichnet eine Blume mit einem Stängel, Blütenblättern und einer Mitte. Kreise (`arc()`), Schleifen (`for`), Formen
`function drawCloud(x, y, size) { ... }` Zeichnet eine Wolke als Gruppe von überlappenden Kreisen. Kreise (`arc()`), Formgruppen
`function drawLandscape() { ... }` Zeichnet den Himmel, die Wolken, die Wiese und die Blumen. Zeichenschichten, Wiederverwendung von Funktionen
`flowers.forEach(...)`
`clouds.forEach(...)`		 Iteriert durch Blumen bzw. Wolken und zeichnet bzw. animiert sie. Arrays durchlaufen, Animationen
`requestAnimationFrame(animateScene);` Fordert den nächsten Animationsframe an (wird von der Browser-Engine gesteuert). Animation, `requestAnimationFrame`
`button.addEventListener('click', () => { ... });` Schaltet die Animation ein/aus, wenn der Button geklickt wird. Event Listener, Interaktivität
`ctx.clearRect(0, 0, canvas.width, canvas.height);` Löscht den gesamten Canvas-Bereich, bevor neue Frames gezeichnet werden. Canvas-Reset, Animation
`flower.x += Math.sin(flower.offset) * flower.sway;`
`flower.offset += 0.05;`		 Simuliert das Schwanken der Blumen durch eine Sinusfunktion. Animation, Sinusbewegung
`cloud.x += cloud.speed;`
`if (cloud.x - cloud.size > canvas.width) { ... }`		 Bewegt die Wolken von links nach rechts; setzt sie zurück, wenn sie den Bildschirm verlassen. Animation, Endlosschleifen
